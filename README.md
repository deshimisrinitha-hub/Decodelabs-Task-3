# DecodeLabs Task 3: IoT Data Monitoring Dashboard 📊

## Project Overview

This project creates a simple IoT monitoring dashboard to display sensor data visually.

The dashboard shows sensor readings in an organized format and helps users monitor IoT data easily.

## Objective

- Create an IoT data monitoring system
- Display sensor readings visually
- Understand data visualization concepts
- Monitor sensor information

## Features

- Displays temperature data
- Displays humidity data
- Updates sensor readings
- Provides a simple monitoring interface
- Visual representation of IoT data

## Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib

## Project Structure

DecodeLabs_Task3/

│── dashboard.py  
│── sensor_data.csv  
│── README.md  

## Working Principle

1. Sensor data is generated or collected.
2. Data is processed using Python.
3. Dashboard displays sensor readings.
4. Users can monitor IoT values easily.
